#### Readme 
A simple app that has 2 components : A server and client. Each new tab at localhost serves as a new client. Can be used as group chat.

### Server

The server runs using NodeJS + Typescript
To start:

1. `cd api`
2. `npm install`
3. `npm start`

### Client

Runs using React + Typescript
To start:

1. cd `gui`
2. `npm install`
3. `npm start`
